# Kasvi Rajwar — Portfolio
https://kasviii.github.io/portfolio2.0/

A multi-page portfolio site with three doors instead of one long scroll: **Hire Me**, **Wanna Play**, and **See It All** — each with its own mood, plus a running **Study Log**.

## Structure
```
index.html    → hub with the three doors
hire.html     → professional / technical projects
play.html     → fun, built-for-the-joy-of-it projects
seeall.html   → everything, in one wandering playground
study.html    → learning-in-public log
style.css     → shared styles
companion.js  → Byte's logic (blinking, eye-tracking, idle states)
assets/       → character art, path icons, logo
```

## Running it
No build step. Open `index.html` in a browser, or serve the folder with any static host (Live Server, GitHub Pages, Vercel).

## Byte
The character following you around the site — blinking, tracking your cursor, peeking at whatever project you hover near in "See It All" — was inspired by **Mochi**, an interactive login-page character concept by [kindi-ui](https://github.com/kindi-ui/MochiCharacter). Byte's artwork is original, generated fresh for this site, but the underlying idea (a reactive companion with mood states tied to what the visitor is doing) started with Mochi. Credit and thanks to kindi-ui for the original concept.
